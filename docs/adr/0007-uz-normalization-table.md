# 0007. UZ-нормализация: единый канон-латиница

## Статус
Accepted

## Контекст
Узбекский ведётся в двух письменностях параллельно (латиница ↔ кириллица: oʻ/gʻ/q/h ↔ ў/ғ/қ/ҳ и т.д.) плюс варианты апострофа (ʻ U+02BB против обычного '). Поиск по name_uz и стабильные хеш-id групп должны совпадать независимо от письменности ввода.

## Решение
Нормализация приводит обе стороны (запрос и значение) к единому канону — латинице. Реализация: явная таблица соответствий кириллица→латиница + унификация вариантов апострофа к ASCII U+0027; диграфы (ch/sh/ng/yo/yu/ya/oʻ/gʻ) получаются из одиночных кириллических букв. RU — короткий путь (lower+trim+collapse+ё→е). Покрытие normalize — 100%, с юнит-тестами на кросс-скрипт пары.

## Последствия
Запрос на любой письменности находит товар, чьё name_uz ведётся на другой. Хеш-id групп стабилен к смене регистра/письменности. Таблица — единственный источник правды нормализации; расширяется явно.

## Альтернативы
Отвергнуто: «замена пары символов» (ad-hoc replace) — не покрывает диграфы и апострофы, промахи на узбекском.
